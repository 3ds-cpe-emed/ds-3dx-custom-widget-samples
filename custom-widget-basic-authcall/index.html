Welcome to the 3DEXPERIENCE WEB SERVICES CPE EMED Workshop

Simple example of using WAFData to call a 3DEXPERIENCE Web Services to get the security contexts available to the user.

  1. Open the widget menu - note that right after loading there are no Preferences available.
  2. Click on the Refresh menu entry
  3. Now check that the Preferences is now available on the widget menu with a list of security contexts.
